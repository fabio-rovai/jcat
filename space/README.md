# JCAT ontology engine

Paste a flat list of terms and get a standards-compliant SKOS taxonomy or OWL ontology.

This Space is a thin front-end over the open-source [`jcat`](https://github.com/fabio-rovai/jcat)
engine, the same engine that powers the live demo at [jcatlabs.com](https://jcatlabs.com).
The core is free forever; curation and managed infrastructure at scale are the product.
